# C Fundamentals

A collection of C programs focused on problem-solving and core programming fundamentals.

## Projects
- caesar — Caesar cipher
- substitution — Substitution cipher
- readability — Coleman–Liau readability index
- credit — Luhn algorithm + card type detection
- cash — Minimum coins (greedy)
- population — Population growth simulation

## How to run
Each folder contains a program you can compile with a standard C compiler.

Example:
Compile:
gcc program.c -o program

Run:
./program
